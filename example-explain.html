    Hyperopt

    1: Create Pipeline w/ Data

    To create a pipeline, you have two options (se our API API or user interface (UX) to create a pipeline.)

    2: Build a Docker

    If you haven't already, you'll need to install the environment. You can do this with pip:

    docker build -t my-image .

    3: Create a Payload

    An example of possible training parameters might be the following (look at the API for more information)

    {
    "next-item": {
        "params": {
           "timeseries": True,
            "x": {},
            "y": {},
            "pipeline": "2890379nvjs"
            "user_secret": "lkjho823hikuwn"
          }
    }
}

    You can call it from python using the following.

    curl -X POST -H "Content-Type: application/json" -d '{"data": {"model": img...}, "meta_data":{}}' http://localhost:5000/v1/explain
